# titanic-ml-project
Titanic Survival Prediction 🚢
 Project Overview

This project predicts whether a passenger survived the Titanic disaster using machine learning techniques.

 Dataset

    The dataset is based on passengers from the Titanic shipwreck and includes features like age, gender, ticket class, and fare.

 Workflow:
 
    Data Cleaning
    
    Exploratory Data Analysis (EDA)
    
    Feature Engineering
    
    Model Training
    
    Model Evaluation
    
    Model Improvement

 Models Used:
 
    Logistic Regression
    
    Random Forest Classifier

 Results:

    Logistic Regression Accuracy: ~81%
    
    Random Forest Accuracy: ~83%
    
    Improved recall for survivors from 0.74 → 0.80


 Key Insights:
 
    Gender is the most important factor (females survived more)
    
    Higher class passengers had better survival rates
    
    Fare strongly correlates with survival
